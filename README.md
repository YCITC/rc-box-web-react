# shuooa-react-project
### Usage
* Clone this project
* Into project folder, install packages  
  ``` npm install``` 
  or 
  ``` yarn install```
* Build:  
  ```npm run build```  or ```yarn run build``` or ```yarn build```
* Run dev server:  
  ```npm run dev``` or ```yarn run dev``` or ```yarn dev```
* Upgrade packages:  
  ```yarn upgrade-interactive --latest ```
